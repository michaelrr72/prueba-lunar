# Prueba Lunar — versión organizada

Estructura lista para GitHub Pages.

## Archivos
- `index.html`
- `assets/css/styles.css`
- `assets/js/data.js`
- `assets/js/app.js`

## Mejoras incluidas
- Separación de HTML, CSS y JS
- Eventos con `addEventListener`
- Persistencia con `localStorage`
- Render seguro de condiciones
- Mantiene la estética y la temática de Genshin Impact

## Despliegue en GitHub Pages
1. Sube estos archivos a tu repositorio.
2. Ve a **Settings > Pages**.
3. En **Source**, elige tu rama principal y la carpeta **/(root)**.
4. Guarda los cambios.
